# PC Helpdesk Support Mini Project

A practical mini-project demonstrating core PC helpdesk and technical support skills.  
This repository includes troubleshooting guides, scripts, sample tickets, and knowledge base articles designed to simulate real-world IT support work.

---

## Project Contents

- Step-by-step troubleshooting guides  
- Useful helpdesk scripts (PowerShell and Batch)  
- Professional knowledge base (KB) documentation  
- Sample support tickets  
- A structured layout that reflects real IT workflows  

---

## Skills Demonstrated
- Windows troubleshooting  
- Network diagnostics  
- User communication  
- Ticketing and documentation  
- Hardware/software issue resolution  
- Automation using scripts
